# 🎓 SkillSphere

SkillSphere is a modern online learning platform where users can explore courses, register, login, and manage their profile.

---

## 🚀 Features

- 🔐 Authentication (Email & Google Login)
- 👤 User Profile Page
- ✏️ Update Profile (Name & Image)
- 📚 Browse Courses
- 🔍 Search & Filter Courses
- ⭐ Popular Courses Section
- 🎨 Clean & Responsive UI

---

## 🛠️ Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Better Auth
- MongoDB
- React Hot Toast
- Swiper js (npm Package)

---

## ▶️ Run Locally

npm install  
npm run dev
